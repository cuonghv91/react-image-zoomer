# react-image-zoomer

## Install

```bash
npm install --save react-image-zoomer
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'react-image-zoomer'
import 'react-image-zoomer/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
